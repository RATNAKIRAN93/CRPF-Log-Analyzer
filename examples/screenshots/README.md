# Screenshots

This directory contains screenshots of the CRPF Log Analyzer UI for documentation and training purposes.

## Contents

Store screenshots here with descriptive filenames:

- `login_screen.png` - Login/authentication page
- `main_dashboard.png` - Primary SOC dashboard
- `discover_view.png` - Log search interface
- `alert_details.png` - Alert investigation view
- `system_health.png` - Infrastructure monitoring

## Guidelines

- Use PNG format for quality
- Include descriptive captions in documentation
- Redact any sensitive information
- Keep screenshots up-to-date with UI changes

## Usage

Reference screenshots in documentation:

```markdown
![Main Dashboard](../examples/screenshots/main_dashboard.png)
```
