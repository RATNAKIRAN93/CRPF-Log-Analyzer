# Backend Scripts

This directory contains automation and utility scripts for the CRPF Log Analyzer.

## Categories

### Setup Scripts
- Agent deployment automation
- Initial configuration scripts
- Environment setup

### Maintenance Scripts
- Index management (create, delete, rotate)
- Backup and restore
- Health checks

### Data Processing
- Log enrichment scripts
- Data migration utilities
- Bulk import/export tools

## Usage

Scripts should be run from the repository root:

```bash
# Example
python backend/scripts/script_name.py
```

## Requirements

Install Python dependencies:

```bash
pip install -r requirements.txt
```
