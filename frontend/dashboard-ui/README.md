# Dashboard UI

This directory is reserved for the custom React-based dashboard UI.

## Status

Currently in planning phase. The primary UI is provided by:
- OpenSearch Dashboards (http://localhost:5601)
- Kibana (for ELK stack deployments)

## Planned Features

- Simplified analyst interface
- Role-specific views
- Custom alert management
- CRPF branding

## Getting Started (Future)

```bash
# Install dependencies
npm install

# Start development server
npm run dev

# Build for production
npm run build
```
